Colour System

https://copic.jp/en/about/color_system/

Colour system composed of three parts.

Colour Family

The first letter in a colour code. eg. *B*04

Blending Group (Colour Saturation)

The first number in a colour code. eg. B*0*4

The scale goes from Saturated -> Unsaturated. So lower number are more saturated/vibrant and larger numbers are more unsaturated/grey.

Intensity Value

The remaining numbers a colour code. eg. B0*4*

The scale goes from light -> dark.
